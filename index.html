⚡ How does the Subscribe Picker work?

  • Auto-detect form — click the yellow button, point to a <form> on the page, and the picker automatically identifies fields: email, consent, SMS, first name, last name, phone, gender — based on keywords in id/name/label
  • Manual picking — for each field click "Pick element" and click the form element; the picker validates the type (you can't pick a <span> instead of an <input>)
  • Smart selector hierarchy — prefers unique name when the ID contains numbers (dynamic IDs); for radio buttons automatically appends [value="..."]; selectors are scoped to the form (e.g. #customer-form input[name="email"])
  • Optional fields — First Name, Last Name, First & Last Name (single field), Phone, SMS checkbox, Female radio, Male radio — only appear in the snippet when picked
  • Generates snippet following a pattern with mousedown + keydown Enter support, localStorage guard prevents double-submit, _edrone.init() fires only when the consent checkbox is checked
  • Live test — injects the snippet, simulates a click, intercepts XHR/fetch to api.edrone.me and reports whether the trace was sent

⚡ Como funciona o Subscribe Picker?
